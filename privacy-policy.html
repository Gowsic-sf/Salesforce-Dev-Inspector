Privacy Policy for Salesforce Dev Inspector

Last Updated: April 20, 2025

Introduction

Salesforce Dev Inspector is a Chrome extension designed to enhance developer productivity within Salesforce Lightning by inspecting and downloading Flow metadata, viewing and editing record details, and managing user information. This Privacy Policy explains how we handle data when you use the extension.

Data Collection and Use

The extension does not collect or store personal data on our servers. All data processing occurs within your browser and Salesforce environment. The following data interactions occur:

  • Cookies: The extension accesses Salesforce session cookies (e.g., sid) to authenticate REST API requests. This enables fetching Flow metadata, record details, and user information (e.g., Name, Email, Alias). Cookies are read only for authentication and are not stored, shared, or used for tracking beyond your Salesforce session.
  • User Data: The extension retrieves Salesforce data via the API, including Flow metadata, record fields (e.g., Account Number, Owner), and user profiles (e.g., ID, Username, Email). This data is displayed in the inspector panel and downloaded as JSON files only upon your explicit request. No data is retained by the extension outside your browser.
  • Storage: Local storage is used to save user preferences, such as the dark mode setting (sf-inspector-dark-mode). This contains no personally identifiable information and is limited to a boolean value.

Data Sharing

We do not sell, trade, or share any data with third parties. All data remains within your Salesforce instance, and downloaded files contain only the data you are authorized to access.

Downloads

The extension allows you to download Flow metadata and record data as JSON files. These downloads are initiated by you and include only Salesforce data you can view. No additional data is appended or collected.

Security

Data is processed locally in your browser using secure Salesforce API calls. We do not have access to your data, and no encryption or transmission to external servers occurs.

User Control

- You control the data displayed and downloaded via the inspector panel.
- You can disable the extension or clear local storage via Chrome’s extension settings to remove saved preferences.
- The extension operates only on Salesforce domains (e.g., *.salesforce.com, *.force.com) and requires your Salesforce session for functionality.

Children’s Privacy

The extension is not intended for users under 13. It is a developer tool for Salesforce professionals.

Changes to This Policy

We may update this Privacy Policy to reflect changes in functionality. Updates will be posted here with a revised date. Continued use of the extension constitutes acceptance of the updated policy.
